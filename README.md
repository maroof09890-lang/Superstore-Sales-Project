# Superstore Sales Dashboard

## Overview
Developed an interactive Sales Dashboard using Power BI to analyze Superstore sales data and generate business insights. The dashboard helps track sales performance, profit trends, and customer segment analysis through dynamic visualizations and KPI metrics.

## Tools & Technologies
- Power BI
- Power Query
- Microsoft Excel

## Dashboard Features
- Interactive KPI Cards
- Sales and Profit Analysis
- Customer Segment Comparison
- Monthly Sales Filtering
- Business Performance Visualization
- Dynamic Charts and Slicers

## Key Insights
- Analyzed overall sales and profit performance
- Compared sales across customer segments
- Identified business trends using visual analytics
- Improved reporting through interactive dashboards
